How Pravah Works

Three agents.
Zero lost context.

Agent 01 / Shadow
👁️
The Shadow
Capture Agent

Sits passively on Slack, Jira, and email via OAuth. Builds a live work-state graph — what's in progress, what's blocked, what decisions were made and why. Always watching. Never interrupting.

Agent 02 / Briefer
⚡
The Briefer
Brief Agent

Triggered at a handoff event. Generates a prioritized, context-rich brief — not a message dump, but an intelligent summary ranked by urgency with explicit action items and open questions surfaced.

Agent 03 / Closer
🔒
The Closer
Follow-up Agent

Monitors if the incoming person acted on critical items. 2 hours pass without a P0 being touched? It nudges. Logs handoff quality over time — building a live institutional memory.
